# 🌐 Perfil de Links Sociais

Este projeto é uma solução para um desafio do Frontend Mentor. O objetivo foi construir uma página de perfil com links sociais, deixando o mais fiel possível ao design proposto.

---

## 📌 Visão Geral

O projeto consiste em uma página de perfil pessoal onde os usuários podem acessar links sociais de forma simples, organizada e responsiva.

---

## ✅ Funcionalidades

- Layout responsivo (mobile e desktop)
- Navegação acessível (compatível com teclado)
- Estrutura semântica em HTML
- Interface limpa e moderna

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox e Responsividade)

---

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Uso de HTML semântico
- Responsividade com Flexbox
- Criação de interações com CSS (hover e focus)
- Organização e limpeza do código
